# Divide and Conquer (Game Jam 14)

## Context
  - A game jam competition entry built under time constraints around a "divide and conquer" theme.

## Goals
  - Deliver a playable game jam entry with dual-stick controls, shooting, and a divide/conquer splitting mechanic.
  - Support multiple enemy types and physics-based gameplay.

## Capabilities
  - The game can be played via pre-built executables.
